<h1>Projeto AOC 1 VA Assembly</h1>

<h2>Informações do Projeto:</h2>

Projeto desenvolvido pelo aluno Eduardo Sampaio, todo o código foi desenvolvido em assembly

<h2>Descrição geral do projeto:</h2>

projeto é implementar um sistema de cadastro de clientes e transações para uma agência bancária. Este sistema de cadastro deve ser operado através de um terminal (prompt/shell) que funciona como um interpretador de comandos de texto, ou seja, o sistema vai ficar constantemente checando por entradas de texto (string) e interpretando o que for recebido a partir de uma lista de comandos que o sistema deve ser capaz de executar. Os comandos que devem ser implementados serão descritos adiante na seção de requisitos de projeto.
